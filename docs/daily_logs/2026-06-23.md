### What I learned
- Pydantic v2 idioms: Annotated[str, StringConstraints(...), Field(...)] is the modern way to stack metadata. StringConstraints handles standard string operations (strip, length) declaratively; a custom BeforeValidator is only needed for logic the built-ins cannot express.
- Error design: the real error space for this API is narrow. Validate that input structurally exists, but let the model judge whether content is meaningful (expressed through confidence). The consumer applies its own business rules to the score, the API stays single-purpose and reusable.
